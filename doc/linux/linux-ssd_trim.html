Check ssd TRIM support

sudo lsblk -D
NAME   DISC-ALN DISC-GRAN DISC-MAX DISC-ZERO
sr0           0        0B       0B         0
sda           0      512B       2G         0
├─sda2        0      512B       2G         0
├─sda3        0      512B       2G         0
└─sda1        0      512B       2G         0

And check the values of DISC-GRAN and DISC-MAX columns. Non-zero values indicate TRIM support.

Continuous TRIM

Using the discard option for a mount in /etc/fstab enables continuous TRIM in device operations:

/dev/sda2  /boot       btrfs  defaults,discard   0  2
/dev/sda1  /boot/efi   btrfs  defaults,discard   0  2
/dev/sda3  /           btrfs  defaults,discard   0  2
