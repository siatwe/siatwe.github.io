Mittels Vererbung kann man verschiedene Hierachiestufen in den Klassen ermoeglichen. Funktionen die in einer Oberklasse programmiert wurden, vererben sich auf die Unterklassen und koennen dort verwendet werden. Um von einer Klasse zu erben, existiert in PHP das Schluesselwort **extends**.

Example

class Oberklasse
{

    function HelloWorld()
    {
        echo "Hello World 
";
    }
}

class Unterklasse extends Oberklasse {

    function HelloWorld()
    {
        //Aufruf der Elternmethode durch parent::
        parent::HelloWorld();
        echo "Hallo Welt 
";
    }
}

class UnterUnterKlasse extends Unterklasse {

    function HelloWorld()
    {
        parent::HelloWorld();
        echo "Bonjour monde 
";
    }
}

$object = new UnterUnterKlasse();
$object->HelloWorld();
