Download

https://www.archlinux.org/download/

Commands

  1. Check internet connection (-c COUNT: Stop after sending COUNT ECHO_REQUEST packets) 
    ping -c 3 google.com
  2. Show network interfaces 
    ip link
  3. Optional: Enable wlan connection 
    wifi-menu
  4. If step 1 fails, try to run dhcpd on your prefered INTERFACE showed in step 2 
    dhcpd INTERFACE
  5. Update the package repository (see: pacman reference) 
    pacman -Syy
  6. Install the reflector package 
    pacman -S reflector
  7. Run reflector (-c: COUNTRY) 
    reflector -c "Germany" -f 12 -l 10 -n 12 --save /etc/pacman.d/mirrorlist
  8. Show hard disks (-l: List the partition tables for the specified devices and then exit) 
    fdisk -l
  9. Run cfdisk on selected hard disk and create a primary, bootable partition (and a swap partition, if needed) 
    cfdisk /dev/sdX
  10. Format the created partition (e. g.: btrfs file system) 
    mkfs.btrfs /dev/sdaXX
  11. Mount generated partition to /mnt 
    mount /dev/sdXX /mnt
  12. Install the base system (-i: Install from existing arch linux) 
    pacstrap -i /mnt base base-devel
  13. Create fstab (-U: Use UUIDs for source identifiers, -p: Avoid printing pseudofs mounts) 
    genfstab -U -p /mnt >> /mnt/etc/fstab
  14. Change root into the new system 
    arch-chroot /mnt /bin/bash
  15. Edit /etc/locale.gen and uncomment prefered language (utf8) 
    nano /etc/locale.gen
  16. Create locale 
    locale-gen
  17. Select timezone (-sf: Make symbolic links instead of hard links, remove existing destination files) 
    ln -sf /usr/share/zoneinfo/CONTINENT/TOWN /etc/localtime
  18. Update the hardware clock 
    hwclock --systohc --utc
  19. Create a hostname 
    echo HOSTNAME > /etc/hostname
  20. Edit /etc/hosts, add your HOSTNAME and some needed informations 
    nano /etc/hosts
    /etc/hosts
127.0.0.1	localhost.localdomain	localhost
::1		    localhost.localdomain	localhost
127.0.1.1	localhost.localdomain	HOSTNAME
  21. Change root password 
    passwd
  22. Install sudo (needed later) and grub (optional: dialog, netctl and wpa_supplicant for wlan network functionality) 
    pacman -S grub sudo dialog netctl wpa_supplicant
  23. Install grub on your hard disk 
    grub-install /dev/sdX
  24. Create grub configuration file (-o: --output=file) 
    grub-mkconfig -o /boot/grub/grub.cfg
  25. Exit the chroot environment 
    exit
  26. Unmount the arch partition (-R, --recursive: Recursively unmount each specified directory) 
    umount -R /mnt
  27. Reboot 
    reboot
  28. Login as root
  29. Create new user (-m: Create home, -g: --gid GROUP, -s: --shell SHELL) 
    useradd -m -g users -G wheel -s /bin/bash USER
  30. Create password for new user 
    passwd USER
  31. Edit /etc/sudoer file and uncomment the following line
    32. nano /etc/sudoer
    %wheel      ALL=(ALL) ALL
  32. Exit root environment 
    exit
  33. Login as USER
  34. Install audio, X and wm packages (e. g.: i3-gaps) 
    sudo pacman -S pulseaudio pulsaudio-alsa alsa-utils xorg xorg-xinit i3-gaps dmenu i3status
  35. Register window manager in ~/.xinitrc 
    echo "exec i3" > ~/.xinitrc
  36. Start X 
    startx
