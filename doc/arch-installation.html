Download

https://www.archlinux.org/download/

Commands

  1. Check internet connection 
    ping -c 3 google.com
  2. Show network interfaces 
    ip link
  3. Optional: Enable wlan 
    wifi-menu
  4. Installation of the reflector package 
    pacman -S reflector
  5. Run reflector 
    reflector -c "Germany" -f 12 -l 10 -n 12 --save /etc/pacman.d/mirrorlist
  6. Show hard disks 
    fdisk -l
  7. Run cfdisk on selected hard disk and create a primary, bootable partition 
    cfdisk /dev/sdX
  8. Format the created partition 
    mkfs.btrfs /dev/sdaXX
  9. Mount generated partition to /mnt 
    mount /dev/sdXX /mnt
  10. Install the base system 
    pacstrap -i /mnt base base-devel
  11. Create fstab 
    genfstab -U -p /mnt >> /mnt/etc/fstab
  12. Chroot into arch dir 
    arch-chroot /mnt /bin/bash
  13. Edit /etc/locale.gen and uncomment prefered language 
    nano /etc/locale.gen
  14. Create locale 
    locale-gen
  15. Select timezone 
    ln -sf /usr/share/zoneinfo/KONTINET/TOWN /etc/localtime
  16. Update the hardware clock 
    hwclock --systohc --utc
  17. Create a hostname 
    echo HOSTNAME > /etc/hostname
  18. /etc/hosts +: % 127.0.1.1 ... hostname
  19. Change root password 
    passwd
  20. Install some additional packages 
    pacman -S grub dialog netcfg wpa_supplicant sudo
  21. Install grub on your hard disk 
    grub-install /dev/sdX
  22. Create grub configuration file 
    grub mkconfig -o /boot/grub/grub.cfg
  23. Exit the chroot environment 
    exit
  24. Unmount the arch partition 
    umount -R /mnt
  25. Reboot 
    reboot
  26. Login with root and password
  27. Create new user 
    useradd -m -g users -G wheel -s /bin/bash USER
  28. Create password for new user 
    passwd USER
  29. Edit visudo file and uncomment wheel ALL=(ALL) ALL
  30. Exit root environment 
    exit
  31. Login as USER
  32. Install audio, X and wm packages 
    sudo pacman -S pulseaudio pulsaudio-alsa alsa-utils xorg xorg-xinit i3
  33. Register window manager in ~/.xinitrc 
    echo "exec i3" > ~/.xinitrc
  34. Start X 
    startx
