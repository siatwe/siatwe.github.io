Download

https://www.archlinux.org/download/

Kommandos

  1. Ueberpruefen ob eine Verbindung zum Internet besteht 
    ping -c 3 google.com
  2. Alle Netzwerk Schnittstellen anzeigen 
    ip link
  3. Optional: Wlan Verbindung herstellen. Sollten dazu propitaere Treiber noetig sein, koennen diese nachtraeglich ab Punkt 12 installiert werden 
    wifi-menu
  4. Installation des reflector Pakets, ein Script zum erstellen optimaler mirrorlists 
    pacman -S reflector
  5. Reflector ausfuehren 
    reflector -c "Germany" -f 12 -l 10 -n 12 --save /etc/pacman.d/mirrorlist
  6. Vorhandene Festplatten anzeigen lassen 
    fdisk -l
  7. Cfdisk auf gewaehlter Festplatte ausfuehren 
    cfdisk /dev/sdX
  8. Formatieren der erstellten Partition 
    mkfs.btrfs /dev/sdaXX
  9. Generierte Partition einhaengen 
    mount /dev/sdXX /mnt
  10. Basis System installieren 
    pacstrap -i /mnt base base-devel
  11. fstab generieren 
    genfstab -U -p /mnt >> /mnt/etc/fstab
  12. Chroot ins Arch Verzeichnis 
    arch-chroot /mnt /bin/bash
  13. /etc/locale.gen edieren und eine Sprache auswaehlen 
    nano /etc/locale.gen
  14. Locale generieren 
    locale-gen
  15. Zeitzone festlegen 
    ln -sf /usr/share/zoneinfo/KONTINET/TOWN /etc/localtime
  16. Hardware Uhr updaten 
    hwclock --systohc --utc
  17. Hostnamen festlegen 
    echo HOSTNAME > /etc/hostname
  18. /etc/hosts um folgende Zeile erweitern: % 127.0.1.1 ... hostname
  19. Roots Passwort aendern 
    passwd
  20. Grub, ggf. Wlan Pakete und sudo installieren 
    pacman -S grub dialog netcfg wpa_supplicant sudo
  21. Grub auf Festplatte installieren 
    grub-install /dev/sdX
  22. Grub Konfigurations Datei erstellen 
    grub mkconfig -o /boot/grub/grub.cfg
  23. Aus der Chroot Umgebung ausloggen 
    exit
  24. Die Arch Partition aushaengen 
    umount -R /mnt
  25. Neustart 
    reboot
  26. Login mit Root und Passwort
  27. Neuen User erstellen 
    useradd -m -g users -G wheel -s /bin/bash USER
  28. Passwort fuer neuen Nutzer erstellen 
    passwd USER
  29. Visudo Datei edieren und %wheel ALL=(ALL) ALL auskommentieren
  30. Root Umgebung verlassen 
    exit
  31. Als USER anmelden
  32. Pakete fuer Audioausgabe, X und ggf. einen Windowmanager installieren 
    sudo pacman -S pulseaudio pulsaudio-alsa alsa-utils xorg xorg-xinit i3
  33. Windowmanager in ~/.xinitrc registrieren 
    echo "exec i3" > ~/.xinitrc
  34. X starten 
    startx
