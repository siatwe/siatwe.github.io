Konfiguration

Die Konfiguration des Kommandos erfolgt in der configure() Methode.

Example

protected function configure()
{
    $this
        // Besimmt den Namen des Kommandos (der Teil nach "bin/console")
        ->setName('app:create-user')

        // Eine kurze Beschreibung des Kommandos, wird angezeigt bei: "php bin/console list"
        ->setDescription('Creates a new user.')

        // Die volle Beschreibung des Kommandos, wird angezeigt wenn das Kommando mit --help ausgefuehrt wird.
        ->setHelp('This command allows you to create a user...')
    ;
}

Execute

Die execute()-Methode hat Zugriff zum in- und output Stream der Konsole. Der Input Stream beinhaltet z. B. Optionen und Argumente. Der Output Stream schreibt Nachrichten in das Konsolenfenster.

protected function execute(InputInterface $input, OutputInterface $output)
{
    // Schreibt mehrere Zeilen mit Zeilenumbruch in die Konsole
    $output->writeln([
        'User Creator',
        '============',
        '',
    ]);

    // Schreibt eine Zeile mit Zeilenumbruch in die Konsole
    $output->writeln('Whoa!');

    // Schreibt eine Zeile ohne Zeilenumbruch in die Konsole
    $output->write('You are about to ');
    $output->write('create a user.');
}
