Installation

Arch Linux

sudo pacman -S docker docker-compose

Debian based

sudo apt-get install docker docker-compose

Docker Group

Add user to docker group:

sudo gpasswd -a user docker

Start docker:

Start the docker system daemon:

sudo systemctl start docker

Enable docker system daemon on startup:

sudo systemctl enable docker

Change into / dir and execute the following: 

docker-compose up --build -d

Chroot into the docker container:

docker-compose exec --user=1000 php sh
