This function immediately returns the pressed key (ascii code).

private function userInput()
{
    system('stty cbreak -echo');
    $stdin = fopen('php://stdin', 'r');
    while (1) {
        $input = ord(fgetc($stdin));

        return $input;
    }

    return null;
}

To get the ascii code, you can return the printed key id, e. g. with print_r(), or look here.

private function userInput()
{
    system('stty cbreak -echo');
    $stdin = fopen('php://stdin', 'r');
    while (1) {
        $input = ord(fgetc($stdin));
        print_r($input);
        return $input;
    }

    return null;
}
