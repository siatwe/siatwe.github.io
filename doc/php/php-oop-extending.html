With the keyword **extends** you can extend methods and variables from the parent class to the its children.

Example

class Parent
{

    function HelloWorld()
    {
        echo "Hello World
";
    }
}

class Children extends Parent {

    function HelloWorld()
    {
        //Execute the parent class with 'parent::'
        parent::HelloWorld();
        echo "Hallo Welt
";
    }
}

class AnotherChildren extends Children {

    function HelloWorld()
    {
        parent::HelloWorld();
        echo "Bonjour monde
";
    }
}

$object = new AnotherChildren();
$object->HelloWorld();
