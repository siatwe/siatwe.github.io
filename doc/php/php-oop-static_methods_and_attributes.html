php oop static methods and attributes

created: 20170510

back

Static methods and attributes don't belong to a specific object, they are globally valid. To execute a static method or access a static attribute, you don't have to create an instance of the belonging class.

Example

class User
{
    public $name;

    //Creates a static array
    public static $allUsers = [];

    //Returns the number of users
    public static function numberOfUsers()
    {
        return count(self::$allUsers);
    }

    public static function createNewUser($name)
    {
        if (strlen($name) < 10) {
            echo "To Short!
";
            return null;
        } else {
            $user = new User();
            $user->name = $name;

            self::$allUsers[] = $user;

            return $user;
        }
    }
}

//Creates a new object with the static method createNewUser()
User::createNewUser("Max Mustermann");
//Returns the number of objects within the class User by the static method numberOfUser()
echo "Number of users: " . User::numberOfUsers() . "
";
User::createNewUser("Simon Attila Weis");
echo "Number of users: " . User::numberOfUsers() . "
";
User::createNewUser("Lisa Kurz");
echo "Number of users: " . User::numberOfUsers() . "
";
