The __conctruct() method will be called, when an object of the class are created.

Example:

class User
{
    public $created;
    public $name;
    public $email;

    public function __construct()
    {
        $this->created = date("d.m.Y H:i:s");
    }
}

$simon = new User();
$simon->name = "Simon Weis";

echo "$simon->created";

In the constructor, you can specify parameters.

Example:

class User
{
    public $created;
    public $name;
    public $email;

    public function __construct($name, $email = "")
    {
        $this->name = $name;
        $this->email = $email;
        $this->created = date("d.m.Y H:i:s");
    }
}

$simon = new User("Simon Weis", "simon@web.de");

echo "$simon->created";
