javascript prevent a dropdown menu from falling out of the window

created: 20170825

back

problem

related css

.dropdown-menu {
    position: absolute;
    top: 100%;
    left: 0;
    z-index: 1000;
}

Solution

javascript

$(document).ready(function () {
    $(".dropdown").on("click", function() {
        setTimeout(function() {
            var d = $(".open").find(".dropdown-menu");
            if ((d.offset().left + d.width()) > $(window).width()) {
                d.addClass("offscreen-menu")
            }
        }, 1)
    })
});

In this case you will need the setTimeout-function, because before the click, there is no dropdown menu for the offset calculation (display: none)! With the css property 'visibility: hidden', the setTimeout-function would become redundant.

css

.offscreen-menu {
    left: auto;
    right: 0;
}
