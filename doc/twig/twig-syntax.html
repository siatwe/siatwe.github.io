twig syntax

created: 20170510

back

{{ variable }}

"Say something...": Writes a variable or a result of an operation into the template.

{% for variable in variables %}

"Do something...": This tag controlls the logic inside the template and is used e. g. for loops

{# What's up? #}

"Comment something...": Things written inside these tags are ignored by the template engine.

Twig also contains filter functions. These functions modify the content before you are able to see it. The following example changes the variable letters into uppercase letters, before rendinger it:

Example

{{ title|upper }}

Another example shows the 'raw' filter. With this filter, things inside, will not be changed or processed:

Example

{{ <h1>Example</h1>|raw }}

Output

<h1>Example</h1>
