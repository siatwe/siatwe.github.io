Die __construct()-Methode wird aufgerufen, sobald ein Objekt der Klasse erzeugt wird.

Example

class User
{
    public $created;
    public $name;
    public $email;

    public function __construct()
    {
        //Erstellt die Klassenvariable bei Erzeugung einer Instanz
        $this->created = date("d.m.Y H:i:s");
    }
}

$simon = new User();
$simon->name = "Simon Weis";

echo "Das Objekt von $simon->name wurde erstellt am $simon->created";

Im Konstruktor kann man Parameter spezifizieren.

Example

class User
{
    public $created;
    public $name;
    public $email;

    //$name ist ein Pflichtfeld bei Erzeugung eines Objekts der Klasse User, $email ist optional...
    //...$created wird automatisch erstellt.
    public function __construct($name, $email = "")
    {
        $this->name = $name;
        $this->email = $email;
        $this->created = date("d.m.Y H:i:s");
    }
}

$simon = new User("Simon Weis", "simon@web.de");

echo "Das Objekt von $simon->name wurde erstellt am $simon->created";
