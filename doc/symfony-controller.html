Ein Controller ist eine selbsterstelle PHP Methode die Informationen des Symfony Request Objekts (Anfrage) liest und ein Response Objekt (Antwort) erstellt und zurueckgibt. Dieses Objekt kann eine HTML Seite, JSON, XML, ein Datei Download, ein Redirect (Weiterleitung), ein Fehler oder alles belieblige sein. Der Controller fuehrt dazu jede erdenkliche Logik aus (bzw. laesst ausfuehren), die noetig ist, den Inhalt einer Seite wie gewollt anzuzeigen.

Example

namespace AppBundle\Controller;

use Sensio\Bundle\FrameworkExtraBundle\Configuration\Route;
use Symfony\Component\HttpFoundation\Response;

class LuckyController
{
    /**
     * @Route("/lucky/number")
     */
    public function numberAction()
    {
        $number = mt_rand(0, 100);

        return new Response(
            'Lucky number: '.$number.''
        );
    }
}
