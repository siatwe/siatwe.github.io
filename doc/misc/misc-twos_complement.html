Introduction

Two's complement is a mathematical operation on binary numbers, but also works with all other numerical systems!

Examples

-10-> 100 -2-> 1100100 -16-> 64 (x) Minuend
-10->  90 -2-> 1011010 -16-> 5A (y) Subtrahend

Example: Binary digit system

1. Step: Ones complement of y, invert all digits:
0100101

2. Step: Twos complement of y, add 1:
0100111

3. Step: Add x and y:
   1100100
 + 0100111
 =========
(1)0001010 -10-> 10 Difference

Why it works?

Given a set of all possible N-bit values, we can assign the lower (by the binary value) half to be the integers from 0 to (2N − 1 − 1) inclusive and the upper half to be −2N − 1 to −1 inclusive. The upper half (again, by the binary value) can be used to represent negative integers from −2N − 1 to −1 because, under addition modulo 2N they behave the same way as those negative integers. That is to say that because i + j mod 2N = i + (j + 2N) mod 2N any value in the set { j + k 2N | k is an integer }  can be used in place of j.

For example, with eight bits, the unsigned bytes are 0 to 255. Subtracting 256 from the top half (128 to 255) yields the signed bytes −128 to −1.

The relationship to two's complement is realised by noting that 256 = 255 + 1, and (255 − x) is the ones' complement of x.
