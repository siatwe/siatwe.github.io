First you have to encode the url: url-encode-decode.com. Then replace the encoded url with (url).

Facebook

<a href="facebook.com/sharer/sharer.php?u=(url)">Share</a>

Twitter

<a href="twitter.com/home/status=(url)">Tweet</a>

Google+

<a href="plus.google.com/share?url=(url)">Like</a>
