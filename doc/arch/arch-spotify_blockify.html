With blockify you can mute commercials. It is available in the AUR as blockify. To have this start and run in the background every time Spotify starts you will need to automate this yourself:

Install spotifyAUR with yaourtAUR

yaourt spotify

Install blockifyAUR with yaourtAUR

yaourt blockify

Custom Spotify script

By placing this script at /usr/local/bin/spotify, it gets preferred to /usr/bin/spotify everytime you start Spotify, so there's nothing else to change and updates won't break it.

#!/bin/sh

spotify=/usr/bin/spotify

if [[ -x $spotify && -x /usr/bin/blockify ]];
then
  blockify &
  block_pid=$!
  $spotify
  trap "kill -9 $block_pid" SIGINT SIGTERM EXIT
fi
