symfony project structure

created: 20170512

back

  • app/
    • Enhaelt Konfigurations Dateien und Vorlagen (Templates). Es gilt, alle Dateien ohne PHP Code, findet man hier.
  • src/
    • Enthaelt den PHP Code.
  • bin/
    • Enthaelt ausfuehrbare Programme/Dateien.
  • tests/
    • Automatisierte Tests z. B. Unit Tests findet man hier.
  • var/
    • Automatisch generierte Dateien wie Logs und Cache werden hier gespeichert.
  • vendor/
    • Enthaelt Drittanbieter Software die ueblicherweise mit Composer installiert wird.
  • web/
    • Der Dokumenten Pfad des Projekts. Alles was von aussen einsehbar sein soll, wird hier abgelegt.
