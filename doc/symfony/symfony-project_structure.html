symfony project structure

created: 20170512

back

  • app/
    • Contains configuration files and templates. It applies, all files without PHP code, can be found here
  • src/
    • Contains the PHP code
  • bin/
    • Contains executable code/files
  • tests/
    • Automated tests, e. g. unit test can be found here
  • var/
    • Automatically generated files such as logs and cache will be saved here
  • vendor/
    • Contains third party software, usually installed with Composer
  • web/
    • The document root of the project. All things that should be visible from the outside, are saved here
