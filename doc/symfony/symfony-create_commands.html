Configuration

The commands configuration are written inside the configure() method.

Example

protected function configure()
{
    $this
        //The name of the command
        ->setName('app:create-user')

        //A short description of the command, will be listened in: php bin/console list
        ->setDescription('Creates a new user.')

        //The complete description of the commando, will be showed after execute it with --help
        ->setHelp('This command allows you to create a user...')
    ;
}

Execute

The execute() method has access to the input and output stream of the terminal. The input stream contains options and/or arguments. The output stream writes messages to the terminal window.

protected function execute(InputInterface $input, OutputInterface $output)
{
    // Writes a bulk of lines with wordwrap into the terminal
    $output->writeln([
        'User Creator',
        '============',
        '',
    ]);

    // Writes a line into the terminal and finish it with one wordwrap
    $output->writeln('Whoa!');

    //Writes a line without wordwrap
    $output->write('You are about to ');
    $output->write('create a user.');
}
